# steingmo's Homebrew tap

```sh
brew install --cask steingmo/tap/tailscale-acl
brew install --cask steingmo/tap/keytype
brew install --cask steingmo/tap/certify
```

| Cask | Description |
| --- | --- |
| `tailscale-acl` | [Tailscale ACL Manager](https://github.com/steingmo/tailscale-acl-manager) — edit, visualize, simulate, and test Tailscale ACL policies offline |
| `keytype` | [KeyType](https://github.com/steingmo/keytype) — paste as keystrokes; types your text where ⌘V is blocked |
| `certify` | [Certify](https://github.com/steingmo/certify) — Let's Encrypt certificates with DNS-01 validation, PEM + PFX export |
